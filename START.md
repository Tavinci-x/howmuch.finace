# How to Start the App

## From a Fresh Terminal

1. **Navigate to the project directory:**
```bash
cd "/Users/christer/CODING APPS/howmuch_finance - cursor"
```

2. **Install dependencies (only needed the first time or after pulling changes):**
```bash
npm install
```

3. **Start the development server:**
```bash
npm run dev
```

4. **Open your browser and go to:**
```
http://localhost:3000
```

## Quick Start (if dependencies are already installed)

```bash
cd "/Users/christer/CODING APPS/howmuch_finance - cursor" && npm run dev
```

## Stop the Server

Press `Ctrl + C` in the terminal to stop the development server.
