# howmuch.finance

A minimalist, privacy-focused finance tracker.

## Features

- **Dashboard**: Track income, expenses, and balance in a unified view.
- **Reports**: Visual breakdown of spending by category and monthly trends.
- **Privacy**: All data stored locally in your browser (IndexedDB).
- **Minimalist Design**: Clean, monochrome aesthetic with receipt-style layout.

## Tech Stack

- Next.js 14
- Tailwind CSS
- Dexie.js (IndexedDB wrapper)
- Recharts
- Lucide React

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
